🌍 world-compute-pkg-mngr

Lightweight Bash package manager for Linux.

📥 How to Install

curl -LO https://dev.world-compute.com/wcpm.sh
chmod +x wcpm.sh
./archpm.sh install neofetch

📦 Available Packages

  • neofetch – A command-line system information tool
  • net-tools – This package includes the important tools for controlling the network subsystem of the Linux kernel
  • htop – interactive process viewer. to view cpu usage ram and process.
  • cmatrix – shows text flying in and out in a terminal like as seen in "The Matrix" movie"
