🌍 World Compute Package Manager

Lightweight Bash package manager for Linux

Made with ❤️ by n0b0dy-arch-btw

📥 How to Install (NEW INSTALLER SCRIPT)

curl -LO https://pkg.world-compute.com/installer.sh
chmod +x installer.sh
wcpm install|uninstall <package>

📦 Available Packages

neofetch

A command-line system information tool

Install: wcpm install neofetch

Uninstall: wcpm uninstall neofetch-uninstall ❌ manual removal required

net-tools

Includes tools for controlling the Linux network subsystem

Install: wcpm install net-tools

Uninstall: wcpm uninstall net-tools-uninstall ❌ manual removal required

htop

Interactive process viewer (CPU/RAM monitor)

Install: wcpm install htop

Uninstall: wcpm uninstall htop-uninstall ✅

cmatrix

Terminal "Matrix" rain animation

Install: wcpm install cmatrix

Uninstall: wcpm uninstall cmatrix-uninstall ✅

🖥️ Distro Compatibility

  • Arch Linux: Built for and tested on Arch. Best experience.
  • Debian/Ubuntu (should work, not tested)
  • Fedora (should work, not tested)
  • Alpine Linux (may work, depends on installed base packages)
  • Any system with Bash and curl
